# Dashboards & Localization

## 1. Ringkasan

Fundara perlu menyediakan grafik, dashboard, dan dukungan dual bahasa agar data finansial mudah dipahami oleh finance, program, manajemen, donor, dan board.

## 2. Dashboard Finansial

Dashboard utama:

```text
Fund Balance Dashboard
Budget vs Actual Dashboard
Donor Utilization Dashboard
Advance Aging Dashboard
Cash Position Dashboard
Bank Reconciliation Dashboard
Grant Burn Rate Dashboard
Program Expense Dashboard
Asset Dashboard
Data Health Dashboard
```

## 3. Grafik

Contoh grafik:

- budget vs actual per project;
- fund utilization per donor;
- cash balance trend;
- advance aging by staff;
- spending by budget line;
- restricted vs unrestricted net assets;
- monthly income and expense;
- program vs admin expense ratio;
- burn rate vs time elapsed;
- asset value by category.

## 4. Prinsip Dashboard

Dashboard harus:

- actionable;
- role-based;
- bisa drill-down;
- tidak hanya visual kosmetik;
- terkait dengan workflow perbaikan;
- menampilkan issue dan risiko.

Contoh:

```text
Advance Overdue Rp120 juta
→ klik
→ daftar staff/project
→ klik advance
→ follow-up liquidation
```

## 5. Dual Bahasa

Bahasa awal Fundara:

```text
Indonesia
English
```

Fundara menerjemahkan:

- menu;
- label field;
- status;
- help text;
- system message;
- report title;
- standard account template;
- workflow state;
- notification template.

Fundara tidak otomatis menerjemahkan data input user seperti:

- nama project;
- keterangan transaksi;
- nama vendor;
- deskripsi activity;
- narasi laporan.
